# 课堂练习：进制转换（exchange）
## 功能简介
输入 1~2 位 16 进制数，输出对应的 10 进制数。
进阶要求：实现 10 进制向 16 进制的转换。
## 使用说明
1. 运行程序后，根据提示输入需要的功能（输入1:Hex to Dec, 输入2:Dec to Hex, 不要回车）。
2. 输入原始数（1~2 位 16 进制数或 10 进制数）。
3. 程序会输出对应的 10 / 16 进制数。
## 学习收获
掌握 16 进制与 10 进制之间的相互转换方法。
熟悉字符串输入、数值解析和格式化输出的实现流程。

# 课堂作业：BIOS 中断调用（bios）
## 功能简介
调用 BIOS 的键盘中断（int 16h）和显示中断（int 10h），获取按键信息并显示对应字符。
程序持续运行，直到用户按下任意 shift 键（左/右）后结束。
练习用 BIOS 中断。
## 学习收获
理解 BIOS 中断与 DOS 中断的区别及其应用场景。
掌握 int 16h（键盘）和 int 10h（显示）中断的调用方法。
学会如何直接与硬件交互，获取和处理底层输入输出。
体会汇编语言对硬件资源的直接控制能力。
