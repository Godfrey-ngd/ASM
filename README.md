#作业题目 1： Hello World 汇编程序

## 功能简介
本程序在 DOS 环境下输出 “Hello World” 字符串，是汇编语言的入门。

主要实现
使用 DOS 中断 21H 的 09H 功能打印字符串。

## 学习收获

# 作业题目 2：按要求打印 ASCII 表
## 基本要求
编写汇编程序，输出 ASCII 表中的小写字母部分（a~z），每行显示 13 个字符。

## 功能说明
程序通过 DOS 中断 21H 的 02H 功能逐字符输出。
每行输出 13 个小写字母，共两行，格式如示例图所示。

## 主要实现思路
循环结构：使用嵌套循环实现逐字符输出，每行 13 个字符。
外层循环控制行数（2 行）。
内层循环控制每行输出 13 个字符。
利用 loop 指令实现循环，并注意保存和恢复 CX 的值。
字符处理：通过修改 DH 寄存器的值，依次输出 ASCII 码 97（'a'）到 122（'z'）。
换行处理：每输出 13 个字符后，插入换行符。

## 通过本作业学习循环结构
熟悉 loop 指令的用法及 CX 计数器的作用。
理解嵌套循环在汇编中的实现方式，掌握循环变量的保存与恢复。
体会条件跳转和循环控制在实际输出任务中的应用

# 作业题目3 求和
## 基本要求
编写汇编程序，计算1+2+⋯+100 的和，并将结果 （“5050” ）打印到屏幕上。

功能拓展说明
用户可输入 1~100 范围内的任意一个正整数 𝑛，程序计算 1+2+⋯+n 的和，并以十进制形式输出结果。
程序支持 DOS 环境下的标准输入输出，结果以字符串形式显示在屏幕上。

## 主要实现思路
输入处理：使用 DOS 0AH 功能读取用户输入（最多 3 位数字）。
ASCII 转数值：将输入的 ASCII 字符串转换为对应的数值𝑛
求和计算：利用公式计算 1 到 𝑛 的和，结果存于 AX。
结果转换：将计算结果转换为十进制 ASCII 字符串。
输出显示：依次输出前缀 “Sum = ” 和结果字符串，并换行。

## 关键数据结构
BUF：输入缓冲区，存储用户输入的数字。
RESULT：输出前缀字符串 “Sum = ”。
OUTBUF：结果输出缓冲区，存储转换后的十进制字符串。
CRLF：回车换行符。

## 注意事项
输入范围限定为 1~100，超出范围可能导致结果异常。
结果转换和输出均采用 DOS 21H 中断服务。
