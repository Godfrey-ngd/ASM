# 作业题目 4：乘法指令和过程调用
## mul.asm —— 九九乘法表输出
### 功能简介
程序通过嵌套循环输出标准的 9×9 乘法表，每行显示一组乘法表达式及结果。
使用 mul 指令进行乘法运算，结果通过自定义过程 print 输出。

### 学习收获
掌握 mul 指令的用法，实现整数乘法运算。
学习过程调用（call 和 ret 指令），理解寄存器在过程调用前后的保存与恢复。
复习嵌套循环结构。
体会 DOS 下字符输出的流程和格式控制。


## check.asm —— 九九乘法表纠错与数据段操作
### 功能简介
程序检查数据段中预设的 9×9 乘法表数据是否正确，若发现错误则输出错误位置。
通过嵌套循环遍历表格，比较实际乘积与表中数据。

### 学习收获
深入理解数据段的定义与访问，掌握数据查找与定位方法。
复习嵌套循环的实现，体会循环变量的使用和边界控制。
进一步掌握过程调用，理解主过程与子过程之间的寄存器资源管理。
